# Raceup

### How To Build
## Linux
```
mkdir build
cd build 
cmake ..
make
./Raceup
```
## On the browser 
An easier way to run the code is to go to an [Online Compiler](https://onlinegdb.com/ooXR8ylU4).
### How it works
It's just a simple BFS inside the the world of the game that is saved from the file as a string.
